## Description

## Technologies Used

With the exponential growth of distributed data generated from IoT devices, cloud platforms, and geographically separated databases, centralized data mining approaches face significant challenges in scalability, privacy preservation, communication overhead, and computational efficiency. Distributed Data Mining (DDM) provides a framework for extracting knowledge without centralizing raw data. This report presents a federated learning-based distributed classification model as a novel solution to the CA-2 problem statement
